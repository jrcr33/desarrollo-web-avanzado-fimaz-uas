
El presente proyecto consiste en el desarrollo de una aplicación web dinámica que implementa un sistema CRUD (Create, Read, Update, Delete) para la gestión de productos, utilizando el lenguaje PHP, la tecnología PDO (PHP Data Objects) para la conexión con la base de datos MySQL, y el paradigma de Programación Orientada a Objetos (POO).

La aplicación permite al usuario interactuar con los datos mediante una interfaz sencilla e intuitiva, en la cual se integra un formulario para la captura de información y una tabla para la visualización de los registros, todo dentro de una sola página.

Este desarrollo busca aplicar buenas prácticas en la organización del código, la reutilización de componentes, la seguridad en el acceso a datos y la separación de responsabilidades mediante una estructura basada en controladores, modelos y configuración.

Objetivo

Desarrollar una aplicación web funcional que permita gestionar productos mediante operaciones CRUD, aplicando principios de programación orientada a objetos y utilizando PDO para garantizar una conexión segura y eficiente con la base de datos.

Justificación

El uso de PDO en conjunto con la programación orientada a objetos permite construir aplicaciones más seguras, escalables y fáciles de mantener. Este proyecto fortalece el entendimiento del manejo de bases de datos en PHP, así como la correcta estructuración de un sistema web basado en buenas prácticas de desarrollo.

Requisitos del sistema

Para la correcta ejecución del proyecto, se requiere:

PHP 8 o superior
Servidor local como XAMPP
Apache y MySQL activos
Navegador web (Google Chrome, Microsoft Edge, etc.)
phpMyAdmin para la gestión de la base de datos
