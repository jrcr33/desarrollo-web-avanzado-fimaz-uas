Práctica 4 - POO en PHP

Objetivo de la práctica
Desarrollar un sistema básico en PHP aplicando Programación Orientada a Objetos, utilizando herencia, validación de datos y manejo de excepciones para controlar errores sin detener la ejecución.

Requisitos
- PHP 8
- XAMPP
- Navegador web

Ruta de ejecución en navegador
Una vez colocado el proyecto dentro de la carpeta `htdocs`, se debe acceder desde el navegador a la siguiente ruta:
http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-4/
