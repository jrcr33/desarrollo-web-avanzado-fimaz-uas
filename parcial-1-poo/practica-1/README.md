Práctica 1: Creación de clases y encapsulamiento en PHP
Objetivo

El objetivo de esta práctica fue aplicar los conceptos básicos de la programación orientada a objetos en PHP mediante la creación de una clase llamada Usuario, utilizando atributos privados, constructor, métodos getters y setters, para comprender mejor el uso del encapsulamiento.

Descripción de la clase creada

La clase Usuario fue diseñada con dos atributos privados: nombre y correo.
Se implementó un constructor para inicializar los valores al crear el objeto.
También se agregaron métodos getters para obtener los datos y setters para modificarlos de manera controlada.
Además, se incorporó una validación simple para evitar que los campos queden vacíos.

 Instrucciones de ejecución

1. Abrir XAMPP e iniciar Apache.
2. Guardar la carpeta practica-1 dentro del directorio correspondiente.
3. Abrir el navegador.
4. Ejecutar el archivo `index.php` desde localhost.
5. Verificar que se muestran los datos iniciales y los datos actualizados del usuario.
